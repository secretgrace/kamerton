# КАМЕРТОН / KAMERTON

> Культурный проект. A cultural project.

Сайт: [secretgrace.github.io/kamerton](https://secretgrace.github.io/kamerton/)

---

## О проекте

**КАМЕРТОН** — это культурное пространство, центром которого является симфоническая музыка. Презентация выполнена в формате цифрового архива — одно непрерывное скроллящееся полотно с фоновой музыкой.

Музыкальная основа: **М. Ипполитов-Иванов — Симфония №1, ми минор, op. 46**
